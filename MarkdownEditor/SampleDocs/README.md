# Welcome to Markdown Editor

A beautiful, minimalistic markdown editor built with **WPF** and **.NET 10**.

## Features

### File Explorer
- Browse your project folders
- Click any `.md` file to open it
- Drag & drop files or folders

### Syntax Highlighting
- Headers are color-coded by level
- **Bold** and *italic* text is highlighted
- `Inline code` stands out
- Links and images are easy to spot

### Document Outline
- Automatically extracts headings
- Click any heading to jump to it
- Updates in real-time as you type

## Getting Started

1. Open a folder using **Ctrl+Shift+O**
2. Click any markdown file in the explorer
3. Start editing!

## Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| Ctrl+N | New File |
| Ctrl+O | Open File |
| Ctrl+Shift+O | Open Folder |
| Ctrl+S | Save |
